###栈##

问题描述：

As the new term comes, the Ignatius Train Station is very busy nowadays. A lot of student want to get back to school by train(because the trains in the Ignatius Train Station is the fastest all over the world ^v^). But here comes a problem, there is only one railway where all the trains stop. So all the trains come in from one side and get out from the other side. For this problem, if train A gets into the railway first, and then train B gets into the railway before train A leaves, train A can’t leave until train B leaves. The pictures below figure out the problem. Now the problem for you is, there are at most 9 trains in the station, all the trains has an ID(numbered from 1 to n), the trains get into the railway in an order O1, your task is to determine whether the trains can get out in an order O2.

1.栈的定义

typedef struct
{
    int *base;
    int *top;
    int size;
}Sqstack;

2.栈的操作

int push(Sqstack &S,int e)
{
    if(S.top - S.base >= S.size)
    {
        S.base = (int *)realloc(S.base,(S.size + 1)*sizeof(int));
        S.top = S.base + S.size;
        S.size += 1;
    }
    *S.top = e;
    S.top++;
    return 0;
}//入栈

int pop(Sqstack &S)
{
    if(S.base == S.top)
    {
        return 0;
    }
    else
    {
        -- S.top;
        return 0;
    }
}//出栈

int boolEmpty(Sqstack S)
{
    if(S.top == S.base)
    {
        return 1;
    }
    else
    {
        return 0;
    }
}//判断栈空

3.解决代码

int main()
{
    Sqstack S;
    int n;
    string str1,str2;
    string a[30],b;
    int *p;
    bool judge = true;
    while(cin>>n)
    {
        S.base=(int *)malloc(10*sizeof(int));
        S.top = S.base;

        cin>>str1;
        //输入O1，进站顺序

            cin>>str2;
        //输入O2，出站顺序
        int m = 0;//记录下一次O1循环开始的位置
        int k = 0;//记录string中的in和out字符串
        a[k ++] = "Yes.";
        for(int i = 0;i < str2.length();i ++)//出站序列O2循环
        {
            p = S.top;
            if(m == n )
                {

                    if(str2.at(i) - '0' == *(--p))
                    {
                        pop(S);
                        a[k ++] = "out";
                    }
                    else
                    {

                        judge = false;
                        break;
                    }
                }
            else
            {
                for(int j = m;j < str1.length();j ++)//进站O1序列循环
                {

                    if(str2.at(i) - '0' == *(-- p))
                    {
                        pop(S);
                        a[k ++] = "out";
                        m += 1;
                        break;
                    }
                    else if(str2.at(i) - '0' == str1.at(j) - '0')
                    {
                        push(S,str1.at(j) - '0');
                        a[k ++] = "in";
                        pop(S);
                        a[k ++] = "out";
                        m += 1;
                        break;
                    }
                    else
                    {
                        push(S,str1.at(j) - '0');
                        a[k ++] = "in";
                        m += 1;
                    }
                    p = S.top;
                }

            }
        }
        if(judge == true)
        {
            for(int i = 0;i < k;i ++)
            {
                cout<<a[i]<<endl;
            }
            cout<<"FINISH"<<endl;
        }
        if(judge == false)
        {
                cout<<"NO."<<endl;
                cout<<"FINISH"<<endl;
        }
    }
    return 0;
}

###链表###

问题：

某部队进行新兵队列训练，将新兵从一开始按顺序依次编号，并排成一行横队，训练的规则如下：从头开始一至二报数，凡报到二的出列，剩下的向小序号方向靠拢，再从头开始进行一至三报数，凡报到三的出列，剩下的向小序号方向靠拢，继续从头开始进行一至二报数。。。，以后从头开始轮流进行一至二报数、一至三报数直到剩下的人数不超过三人为止。

1.链表结构的定义

typedef struct LNode
{
    int data;
    struct LNode *next;
}*LinkList;

2.链表的初始化

int Init(int m,LinkList &head,LinkList &pre,LinkList &tmp)
{
    pre = head = tmp =(LinkList)malloc(sizeof(LNode));
        tmp->data = 1;
        tmp->next = NULL;
        for(int i = 2;i <= m;i ++)
        {
            tmp->next=(LinkList)malloc(sizeof(LNode));
            tmp->next->data = i;
            tmp->next->next = NULL;
            tmp = tmp->next;
        }
    return 0;
}

夏末秋初--写在八月尾巴上的留恋

发表于 2016-09-01   |   分类于 生活

八月的最后几个小时里，漫步在校园里。电话那头是高中死党熟悉的声音，相互询问着开学的日期，随之而来的便是一阵长吁短叹。时间何其的相似，人也还是那人，景色也还是那景色，就连心情也似乎没有太多的变化。

学校里的杨树已然开始落叶，干黄枯萎的树叶散落了一地，在人们路过时，会发出细微的碎裂的声音。古人云“一叶落而知秋”，也正因此才真切的感受到秋天的到来，而又一个夏天悄悄的，悄悄的又走远了。每当此时，也才会忽然想起，夏天来临之前，那些对自己说好的要做的事又一次的爽约了。

对于夏天有种莫名的喜爱，大概是因为高中的时候青春杂志看多了吧，潜移默化的，便在心里觉得，夏天就应该是洋溢着青春的季节，就像是情侣手中冰块和气泡混合着的柠檬水。

电话里，死党有了女票，是我们都很熟相识了很久的女同学。起初很惊讶，诧异于画风差异这么大两个人竟然走到了一起。可是慢慢的，又觉得他们俩莫名的合拍。看着他们俩在社交圈里的互动，心里竟有种幸福的感觉，真心祝福他们！

大多数的情况下，我都是作为一个旁观者来审视爱情的，当然，我不反感旁观者的身份，只是，偶尔也想从这个身份里跳脱出来。

整个八月，基本上都是在学校里度过的，没有自己想象中的那么认真学习，原定的很多学习计划也都没有能完全实现，可也还算是收获了很多吧！临近开学，回校的人越来越多，心里莫名的有种失落。就好像心爱的东西被人夺走了一样，原本属于我的安静的校园被打破了！

就先写这么多吧，心里牵扯的事情太多，想要静下心来写点东西真的很困难，原谅我毫无头绪，胡言乱语的一通话，晚安！
